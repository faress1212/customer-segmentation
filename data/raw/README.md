# Data

This project uses the **Mall Customers Segmentation** dataset.

Download it from Kaggle:
https://www.kaggle.com/datasets/abdallahwagih/mall-customers-segmentation

**Dataset summary:**
- 200 customer records
- Columns: `CustomerID`, `Gender` (or `Genre`), `Age`, `Annual Income (k$)`, `Spending Score (1-100)`
- No missing values in the original dataset
- Commonly used as a beginner-friendly dataset for clustering (KMeans, DBSCAN, Hierarchical Clustering)
