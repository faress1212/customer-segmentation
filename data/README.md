# Data

المشروع بيستخدم داتاسيت **Mall Customers Segmentation** من Kaggle.

حمّلها من هنا:
https://www.kaggle.com/datasets/abdallahwagih/mall-customers-segmentation

بعد التحميل، حط ملف `Mall_Customers.csv` هنا بالظبط:
```
data/raw/Mall_Customers.csv
```

**نبذة عن الداتاسيت:**
- 200 عميل لمول
- الأعمدة: CustomerID, Gender, Age, Annual Income (k$), Spending Score (1-100)
- مفيش عمود "إجابة صح" (Unsupervised) — المطلوب اكتشاف مجموعات العملاء بنفسنا
